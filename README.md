# theinternetsaga

## Dependancies
- Middleman 3.4.0
- Sass 3.4.19
- Bourbon 4.2.6
- Bitters 1.1.0

## Getting Started
1. `git clone https://github.com/gcherubin/gc_portfolio.git`
2. `cd gc_portfolio`
3. `rm -rf CHANGELOG.md`
4. `bundle exec middleman server`
